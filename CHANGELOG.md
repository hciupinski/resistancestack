# Changelog

All notable changes to ResistanceStack are documented in this file.

The format follows Keep a Changelog, and releases use semantic versioning.

## [0.1.1] - 2026-05-05

### Added

- Public OSS release workflow with GitHub Actions CI, security checks, GoReleaser publishing, checksums, and Homebrew tap updates.
- Linux and macOS binary archives for amd64 and arm64 through GoReleaser.
- Security reporting policy and GitHub issue templates.
- CLI version metadata for released binaries.

### Included

- Cobra/Viper CLI, local inventory and audit modes, doctor preflight checks, guided wizard, HTML reports, safer deploy-user bootstrap, host rollback, deployment profiles, and safe adoption documentation.

## [0.1.0] - 2026-05-04

### Added

- Initial internal release baseline for the ResistanceStack MVP.
